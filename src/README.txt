This directory contains the source files to create the LOST web application.
----------------------------------------------------------------
The folowing files are located in the src directory:

1. app.py 		-python script that runs with python3. Uses flask to create a web application for LOST

2. lost_config.json	-JSON formatted file. Used to configure the LOST application

3. config.py		-python script that runs with python3. Parses lost_config.json so programs can read the info

4. templates 		-directory.  Holds html files that are templates for the LOST web application
